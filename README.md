# String

Project with three main functions:

1- Convert string to all caps
Ex: `Hello Let's play LoL` => `HELLO LET'S PLAY LOL`

2- Convert string to caps alternate
Ex: `Hello Let's play LoL` => `HeLlo LeT's PlAy LoL`

3- Convert string to low caps
Ex: `Hello Let's play LoL` => `hello let's play lol`
